Components

Key React components in the Enclavr frontend.

Chat Components

  • Chat.tsx - Main chat container
  • ChatInput.tsx - Message input field
  • MessageItem.tsx - Individual message display

Voice Components

  • VoiceChat.tsx - Voice chat container
  • MediaRecorder.tsx - Audio/video recording

DM Components

  • DMChat.tsx - Direct message chat

Room Components

  • RoomCard.tsx - Room display card
  • CategoryCard.tsx - Category display card
  • CreateRoomModal.tsx - Create room dialog
  • CategoryModal.tsx - Category management
  • JoinRoomModal.tsx - Join room dialog

User Components

  • PresenceList.tsx - Online users list
  • MembersModal.tsx - Room members list

Moderation Components

  • BanList.tsx - Ban management
  • BanDialog.tsx - Ban user dialog
  • ReportsPanel.tsx - Report queue
  • ReportDialog.tsx - Report user dialog

Feature Components

  • FileUpload.tsx - File upload handler
  • FileList.tsx - Uploaded files list
  • InviteModal.tsx - Invite system
  • SettingsModal.tsx - User settings
  • SearchModal.tsx - Search interface
  • WebhookModal.tsx - Webhook management
