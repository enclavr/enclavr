Commands

Development commands for Enclavr components.

Frontend Commands

# Install dependencies
bun install

# Start development server
bun run dev

# Run unit tests
bun run test:run

# Run E2E tests
bun run test:e2e

# Lint code
bun run lint

# Build for production
bun run build

Server Commands

# Run the server
go run ./cmd/server

# Run tests
go test -v ./...

# Run with specific test
go test -v -run TestName ./...

# Build binary
go build -o bin/server ./cmd/server

# Run linter
golangci-lint run ./...

Infrastructure Commands

# Start all services
docker-compose up -d

# Stop all services
docker-compose down

# View logs
docker-compose logs -f

# Rebuild services
docker-compose up -d --build
