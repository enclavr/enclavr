Database Models

PostgreSQL database models using GORM.

Core Models

  • User - User accounts
  • Room - Voice/text chat rooms
  • Category - Room categories
  • Message - Chat messages
  • DirectMessage - DMs between users

Feature Models

  • Reaction - Message reactions
  • Thread - Message threads
  • Poll - Message polls
  • PinnedMessage - Pinned messages
  • Invite - Room invites

Moderation Models

  • Ban - User bans
  • Report - User reports
  • AuditLog - Action logs

Relation Models

  • RoomMember - Room membership
  • UserRole - Role assignments

Integration Models

  • Webhook - Webhook configs
  • PushSubscription - Push notifications
