Rooms API

Endpoints for room management.

Get All Rooms

GET /api/rooms
Authorization: Bearer <token>

Get Room

GET /api/room?id=<room-id>
Authorization: Bearer <token>

Create Room

POST /api/room/create
Authorization: Bearer <token>
Content-Type: application/json

{
  "name": "string",
  "description": "string",
  "password": "string",
  "categoryId": "uuid"
}

Join Room

POST /api/room/join
Authorization: Bearer <token>
Content-Type: application/json

{
  "roomId": "uuid",
  "password": "string"
}

Leave Room

POST /api/room/leave
Authorization: Bearer <token>
Content-Type: application/json

{
  "roomId": "uuid"
}
