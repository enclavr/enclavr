API Client

The API client module structure.

Main API Module

src/lib/api.ts contains the main API client with methods for:

  • Authentication (login, register, refresh)
  • Rooms (create, join, leave, list)
  • Messages (send, edit, delete, search)
  • Direct Messages
  • Voice (ICE config)
  • Admin (bans, reports, roles)

Modular API Structure

The API is split into modules:

  • src/lib/api/base.ts - Base client with auth headers
  • src/lib/api/auth.ts - Authentication endpoints
  • src/lib/api/room.ts - Room management
  • src/lib/api/chat.ts - Messaging
  • src/lib/api/dm.ts - Direct messages
  • src/lib/api/presence.ts - User presence
  • src/lib/api/admin.ts - Admin endpoints

Usage Example
