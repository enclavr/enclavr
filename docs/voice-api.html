Voice API

Endpoints for WebRTC voice communication.

Get ICE Configuration

GET /api/voice/ice
Authorization: Bearer <token>

Response

{
  "iceServers": [
    {"urls": "stun:stun.l.google.com:19302"},
    {"urls": "turn:turn.example.com:3478", "username": "user", "credential": "pass"}
  ]
}

Voice WebSocket

Connect to /api/voice/ws?room_id=<room-id> for real-time voice communication.

WebSocket Messages

TypeDescription
voice_offerSDP offer for peer connection
voice_answerSDP answer for peer connection
voice_iceICE candidate exchange
voice_muteUser muted/unmuted
voice_joinUser joined voice
voice_leaveUser left voice
