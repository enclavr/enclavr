Features

Enclavr is a comprehensive voice chat platform with the following features:

Core Features

Voice Communication

  • WebRTC-based peer-to-peer voice calls
  • Screen sharing support
  • Video support for face-to-face communication
  • Mute/unmute controls
  • TURN server support for NAT traversal

Text Chat

  • Real-time messaging via WebSocket
  • Message reactions with emoji
  • Thread support for conversations
  • for group decisions
  • Message search functionality
  • File uploads support

Direct Messages

  • Private conversations between users
  • Message history persistence

User Presence

  • Online/away/busy/offline status
  • Room presence indicators

Moderation Features

Role-Based Permissions

  • Owner - Full server control
  • Admin - Manage rooms and users
  • Moderator - Enforce rules
  • Member - Standard access

Moderation Tools

  • Ban system - Room and server bans
  • Reports - User reporting workflow
  • Audit logs - Track user actions

Additional Features

Customization

  • Custom emoji per server
  • Stickers support
  • Soundboard with audio clips
  • Room categories organization

Integrations

  • Webhooks for event notifications
  • Push notifications support

Analytics

  • Daily activity tracking
  • User engagement metrics
  • Channel statistics
