Admin API

Endpoints for moderation and administration.

Bans

Create Ban

POST /api/ban
Authorization: Bearer <token>
Content-Type: application/json

{
  "userId": "uuid",
  "roomId": "uuid",
  "reason": "string",
  "expiresAt": "timestamp"
}

Get Room Bans

GET /api/ban/room?room_id=<room-id>
Authorization: Bearer <token>

Check User Ban

GET /api/ban/check?user_id=<user-id>&room_id=<room-id>
Authorization: Bearer <token>

Delete Ban

DELETE /api/ban/delete?id=<ban-id>
Authorization: Bearer <token>

Reports

Create Report

POST /api/report
Authorization: Bearer <token>
Content-Type: application/json

{
  "reportedUserId": "uuid",
  "roomId": "uuid",
  "reason": "string",
  "messageId": "uuid"
}

Get Reports

GET /api/reports?status=pending
Authorization: Bearer <token>

Review Report

PUT /api/report/review?id=<report-id>
Authorization: Bearer <token>
Content-Type: application/json

{
  "status": "resolved",
  "action": "string"
}
