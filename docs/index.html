Enclavr Documentation

Welcome to the Enclavr documentation. Enclavr is a self-hosted voice chat platform, providing real-time voice communication, text chat, direct messages, and more.

Quick Links

Architecture Overview Frontend Docs Server Docs Infrastructure API Reference

Key Features

  • Real-time voice communication using WebRTC
  • Text chat with reactions, threads, and polls
  • Direct messages between users
  • User presence tracking (online/away/busy)
  • Role-based permissions for moderation
  • OIDC authentication support
  • Self-hosted deployment with Docker
