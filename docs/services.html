Services

Docker services in the Enclavr stack.

Frontend

Next.js application serving the web interface.

image: enclavr/frontend:latest
ports: ["3000:3000"]

Server

Go backend API server.

image: enclavr/server:latest
ports: ["8080:8080"]

PostgreSQL

Database for persistent storage.

image: postgres:15-alpine
ports: ["5432:5432"]
volumes: [postgres_data:/var/lib/postgresql/data]

Redis

Pub/Sub and caching.

image: redis:7-alpine
ports: ["6379:6379"]
command: redis-server --appendonly yes

Coturn

TURN server for WebRTC NAT traversal.

image: coturn/coturn:4.6.2-alpine
ports: ["3478:3478", "49152-49172:49152-49172"]

Health Checks

All services include health checks for container orchestration.
