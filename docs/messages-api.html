Messages API

Endpoints for room messaging.

Get Messages

GET /api/messages?room_id=<room-id>&limit=50&before=<message-id>
Authorization: Bearer <token>

Send Message

POST /api/message/send
Authorization: Bearer <token>
Content-Type: application/json

{
  "roomId": "uuid",
  "content": "string"
}

Update Message

PUT /api/message/update?message_id=<message-id>
Authorization: Bearer <token>
Content-Type: application/json

{
  "content": "string"
}

Delete Message

DELETE /api/message/delete?message_id=<message-id>
Authorization: Bearer <token>

Search Messages

GET /api/messages/search?q=<query>&limit=20
Authorization: Bearer <token>

Reactions

Add Reaction

POST /api/reaction/add
Authorization: Bearer <token>
Content-Type: application/json

{
  "messageId": "uuid",
  "emoji": "string"
}

Remove Reaction

DELETE /api/reaction/remove
Authorization: Bearer <token>
Content-Type: application/json

{
  "messageId": "uuid",
  "emoji": "string"
}
