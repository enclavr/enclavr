Testing

Testing strategies for Enclavr.

Frontend Testing

Frontend uses Vitest for unit tests and Playwright for E2E tests.

# Run unit tests
bun run test:run

# Run E2E tests
bun run test:e2e

# Run tests in watch mode
bun run test

Test Structure

  • src/test/ - Unit tests
  • e2e/ - End-to-end tests

Server Testing

Server uses Go's testing package with SQLite in-memory for tests.

# Run all tests
go test -v ./...

# Run specific package
go test -v ./internal/handlers/...

# Run with coverage
go test -v -cover ./...

Test Structure

  • internal/handlers/*_test.go - Handler tests
  • internal/websocket/*_test.go - WebSocket tests

Running All Tests

# Frontend
cd frontend && bun run test:run && bun run test:e2e

# Server
cd server && go test -v ./...
