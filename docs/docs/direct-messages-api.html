Direct Messages API

Endpoints for direct messaging between users.

Get Conversations

GET /api/dm/conversations
Authorization: Bearer <token>

Get DM Messages

GET /api/dm/messages?user_id=<user-id>&limit=50
Authorization: Bearer <token>

Send DM

POST /api/dm/send
Authorization: Bearer <token>
Content-Type: application/json

{
  "recipientId": "uuid",
  "content": "string"
}

Delete DM

DELETE /api/dm/delete?message_id=<message-id>
Authorization: Bearer <token>
