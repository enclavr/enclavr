Infrastructure Overview

Docker Compose based deployment for Enclavr.

Architecture

Enclavr uses Docker Compose for local development and production deployment.

Services

  • Server - Go backend API
  • Frontend - Next.js web app
  • PostgreSQL - Database
  • Redis - Pub/Sub & caching
  • Coturn - TURN server for WebRTC

Quick Start

cd infra
cp .env.example .env
docker-compose up -d
