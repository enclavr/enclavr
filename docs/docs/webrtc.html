WebRTC

WebRTC implementation for voice communication.

useWebRTC Hook

Main hook for managing WebRTC connections.

const {
  localStream,
  peers,
  joinVoice,
  leaveVoice,
  mute,
  unmute,
} = useWebRTC(roomId, userId);

Features

  • Peer-to-peer audio/video
  • Screen sharing
  • Mute/unmute controls
  • ICE candidate exchange
  • STUN/TURN server support

WebRTC Utilities

src/lib/webrtc/ contains:

  • types.ts - TypeScript interfaces
  • ice.ts - ICE server configuration
  • peer.ts - Peer connection management

Signaling

WebRTC signaling is done through the WebSocket connection at /api/voice/ws.
